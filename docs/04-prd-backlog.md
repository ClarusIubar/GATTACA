# 04. 제품 요구사항과 백로그

## 핵심 기능

- 이벤트 등록: 언제, 어디서, 무엇을, 어떻게 할지 기록
- 이벤트 목록/상세 조회
- 메모리 등록: 사진과 캡션 기록
- 코멘트 등록/수정
- 운영자 승인/반려
- 운영자 삭제
- Worker 경유 Kakao relay

## 보조 기능

- demo 권한 전환 모드
- 이벤트/메모리/코멘트 수정
- Cloudflare Pages SPA 라우팅 대응
- runtime-status 기반 배포 상태 확인

## 미래 후보 기능

- 카카오 공유 링크 생성 UI
- 다중 관리자
- 참여자별 활동 통계
- 스토리지 업로드 진행률

## 이번 버전 제외 기능

- 카카오톡 투표/캘린더 API 자동 연동
- 단톡방 멤버 자동 판별
- 푸시 알림

## 비기능 요구사항

- 성능: Cloudflare Pages 정적 배포 기준 첫 진입이 빠르게 렌더링되어야 함
- 보안: 승인 사용자만 CRU, 운영자만 delete
- 접근성: 모바일 대응, 기본 라벨/버튼/폼 접근성 확보
- 유지보수성: 문서와 코드 구조를 일치시켜야 함
- 확장성: Kakao relay 경계와 Cloudflare Worker 구조를 유지한 채 확장 가능해야 함
